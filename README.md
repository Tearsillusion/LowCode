# webTeam
```
该项目采用vue3+ts开发的低代码平台，目前支持基本的按钮，输入框，图片等组件，
组件之间支持复制，删除，撤销，还原等等，具体功能还请自行下载体验吧。

```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
